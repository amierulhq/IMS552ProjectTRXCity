Strategic Solutions

Generate & Evaluate Alternatives

Based on the detailed analysis of the weaknesses and challenges faced by the IT Asset Management System, two solution options were formulated to overcome the issues of data isolation and reliance on manual processes.

Chosen Strategy

Alternative 1: API System Integration

Implement automatic system integration using API (Application Programming Interface) programming between the IT Asset Management System and key corporate systems such as the Employee Management System (ESS) and the Financial System (IFCA).

Advantages
  • Eliminates data silos completely with real-time database synchronization.
  • Eliminates the need for manual data entry by IT staff.
  • Drastically reduces the risk of human error, ensuring accurate ownership records.
Disadvantages
  • Requires a relatively high initial investment cost to appoint a software vendor.
  • Needs a longer testing phase period to ensure safe interdepartmental data transfers.

Alternative 2: Smart Barcode Mobile App

Upgrade the existing physical scanning infrastructure to use smart barcodes linked to a special mobile application. IT staff can use their smartphones to update asset locations directly to the cloud database instead of static scanning machines.

Advantages
  • Much lower implementation cost compared to API integration.
  • Easy to develop and deploy in a very short time framework.
  • Does not involve major changes to the main system code structure.
Disadvantages
  • Does not solve the basic problem of data isolation with other departments.
  • Still requires manual communication with the HR Department.
  • Relies entirely on the discipline of staff to perform scans during asset movement.
Final Decision

Implementation of Alternative 1:
API System Integration

Based on an in-depth evaluation, we recommend that TRX City Sdn Bhd choose the first alternative. This option is seen as the most effective long-term solution because it directly solves the root cause of the data isolation (data silo) problem faced by the IT Department today. By integrating the IT Asset Management System with the main corporate system, the IT Department no longer needs to rely on manual communication.

The Three Pillars of Impact

Why API Integration is critical for supporting TRX City Sdn Bhd's status as a major developer.

Operational Efficiency

Every piece of information regarding the entry of new employees or employees who have resigned will be automatically updated into the asset management database. This eliminates human negligence issues such as delays in recording the status of the company's laptop logins.

Security & Risk Management

When connected to the HR system in real time, digital access rights for critical assets (like servers and network devices) can be revoked immediately upon an employee's termination. This drastically reduces the risk of internal data breaches caused by dormant accounts.

Financial ROI

Although requiring a larger initial financial investment, the long-term ROI is highly worthwhile. Automatic coordination with the Financial System (IFCA) allows asset depreciation calculations, hardware insurance claims, and software license renewals to be done automatically without calculation errors.

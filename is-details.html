Details of Selected Information System

An in-depth evaluation of the IT Asset Management System architecture.

System Overview & Background

For this section, the selected information system to be analysed is the IT Asset Management System. This system is used by the IT Department to collect, manage, monitor, and update information related to physical IT assets owned or used by TRX City Sdn. Bhd.

These assets include laptops, personal computers, servers, network equipment, and data centre resources. The system plays an important role in ensuring that IT assets are properly recorded, traceable, and maintained throughout their lifecycle.

System Classification: In terms of information system classification, the IT Asset Management System can be categorised as an Operational Support System.

Functional Description

The main functions of the system are asset registration, barcode scanning, asset tracking, check-in/check-out control, warranty monitoring, reporting, and data recovery.

Asset Registration

The system allows the IT Department to register IT assets into the asset inventory. When a new asset is received, important information can be recorded, such as asset type, barcode number, location, department, user assignment, warranty status, and other relevant details.

Hardware & Software Configuration

Infrastructure Configuration

Based on the interview findings, the system is supported by IT infrastructure such as servers, network equipment, and data centre resources. Barcode identification helps the IT Department perform asset audits more efficiently.

In terms of software configuration, the system uses an asset management platform to maintain the centralised inventory. The system also depends on database and backup functions.

Security Architecture

TRX City Sdn. Bhd. uses security measures such as firewall protection to detect penetration attempts.

  • Access for external guests is managed through integrated WiFi arrangements.
  • Email passwords are changed every 90 days to strengthen account security.
  • System updates are checked regularly to maintain system stability and reduce technical problems.
